핵심 구현 흐름

발표용으로 설명하기 쉬운 사용자 가치 흐름입니다.

1. 관광지 조회

지역과 컨텐츠 유형으로 여행 후보를 찾고 지도에서 확인합니다.

2. 계획 연결

검색 결과가 내 여행계획에 이미 포함됐는지 자동으로 보여줍니다.

3. 일정 추가

상세보기와 추가 action을 분리하고, 추가할 여행계획을 modal에서 고릅니다.

4. 후기 확인

관광지 상세에서 Hotplace 사진과 방문 후기를 함께 확인합니다.

5. 날씨 점검

여행계획 상세에서 기간별 날씨를 보고 일정을 점검합니다.
