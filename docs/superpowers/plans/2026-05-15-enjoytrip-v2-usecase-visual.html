Diagram Notes

공개 기능

관광지 조회, 관광지 상세, Hotplace 조회, 게시글 조회, 공지사항 조회는 비로그인 상태에서도 접근 가능합니다.

인증 필요

여행계획, Hotplace 등록/수정/삭제, 게시글 등록/수정/삭제는 로그인 상태 확인을 포함합니다.

권한 확인

수정/삭제는 작성자 권한 확인을 포함하며, 공지사항 관리는 관리자 권한 확인을 포함합니다.

제외 범위

F06 관광지 관련 뉴스 크롤링은 이번 WBS 범위에서 제외했으므로 Use Case Diagram에서도 제외했습니다.
